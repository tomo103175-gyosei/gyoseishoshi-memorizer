# 行政書士試験向け 条文暗記・音声読み上げアプリ（プロトタイプ）

行政書士試験の主要3科目（日本国憲法、民法、行政法）の試験対策に特化した、スマートフォン対応の条文暗記・音声読み上げWebアプリケーションです。

---

## 🌟 主な特徴

1. **e-Gov法令検索API v2 との完全連携**
   - デジタル庁公式の最新法令データをリアルタイムに自動フェッチして表示します。AIハルシネーション（嘘の条文）や省略・要約は一切ありません。一字一句正確なインプットが可能です。
   
2. **高速ローカルキャッシュ**
   - 民法（1050条文）などの巨大データも、初回取得時にサーバーにJSONキャッシュ保存されるため、2回目以降はスマホ上でも瞬時に（0.1秒以下で）ロードされます。

3. **一文単位の精密音声読み上げ & 同期ハイライト**
   - 条文を「。 （句点）」で自動分割し、音声合成エンジン（SpeechSynthesis）と連携して**一文ずつ読み上げ**ます。
   - モバイルブラウザで発生しがちな「長文での音声エンジンフリーズ」を完全に回避しています。
   - 現在読んでいる一文が美しいイエローで発光し、自動的に画面がスクロール追従するため、目で追うのが非常に楽です。
   - 画面上の任意の文章をタップすることで、そこからすぐに読み上げを開始できます。

4. **高度な音声コントロール**
   - 「再生」「一時停止」「停止」を完全サポート。
   - **0.5倍速〜3.0倍速まで0.1刻みで可変できる再生速度コントロール**を搭載（1.0x / 1.5x / 2.0x のワンタップ切替ボタンも完備）。
   - **連続自動再生（ハンズフリー学習）**：1つの条文が終わると自動で次の条文に進んで読み上げるため、通勤電車の中や家事をしながらの「耳学」が完璧に行えます。

5. **集中読解モード (Focus Mode)**
   - 「集中読解」トグルをONにすると、ナビゲーションやヘッダーが非表示になり、特大のフォントサイズと十分な余白、行間のみの超シンプルなインプット画面に変貌します。

## 🚀 起動方法

ローカルのPC環境でアプリを起動するには、以下の手順を実行してください。

1. **依存パッケージのインストール** （既に実行済みです）
   ```bash
   npm install
   ```

2. **サーバーの起動**
   ```bash
   npm start
   ```

3. **ブラウザでアクセス**
   ブラウザを開き、以下のURLにアクセスしてください。
   ```
   http://localhost:3000
   ```
   *スマートフォンで確認する場合は、PCと同じローカルWi-Fiに接続し、PCのプライベートIPアドレス（例：`http://192.168.1.XX:3000`）でアクセスしてください。*

## 📂 プロジェクト構造

* `server.js` - Expressサーバー & e-Gov API v2からJSON形式の法律データを取得し、編・章・節・条・項・号が整理されたフラットな構造に自動解析・パースしてキャッシュするバックエンドスクリプト。
* `public/index.html` - スマホ特化の画面構成を定義するSPA用HTML。
* `public/style.css` - 目が疲れないダークグラスモーフィングデザインと、一文ハイライト等のアニメーションを定義するスタイルシート。
* `public/app.js` - Web Speech API (TTS) 一文単位再生制御、自動スクロール追従、連続再生ステートマシンを含むクライアント制御ロジック。
* `data/cache/` - パース済みの軽量JSONファイルが保存されるキャッシュフォルダ。
